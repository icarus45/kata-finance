Comment configurer ce projet ?
 1 - init configuration

 forexFirstLineNumber       : indique la position à partie de laquelle la lecture des données commence dans le fichier forex.csv, la premier ligne est la ligne 1
 pricesFirstLineNumber      : indique la position à partie de laquelle la lecture des données commence dans le fichier prices.csv, la premier ligne est la ligne 1
 productFirstLineNumber     : indique la position à partie de laquelle la lecture des données commence dans le fichier product.csv, la premier ligne est la ligne 1
 forexLineSeparator         : indique le separateur utilisé pour le fichier Forex.csv
 pricesLineSeparator        : indique le separateur utilisé pour le fichier Prices.csv
 productLineSeparator       : indique le separateur utilisé pour le fichier Product.csv
 reportingPortfolioFileName : nom du fichier contenant les protefeuilles
 reportingClientFileName    : mon du fichier contenant les produits clients
 inputFileSeperator         : indique le separateur utilisé pour les fichiers en output
